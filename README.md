# Smart Assistant V3 🤖

An AI Personal Assistant built in Python with modular architecture.

## Features
- Task Management (CRUD)
- Notes system
- Persistent storage using JSON
- Logging system
- Command history tracking
- Modular design using packages

## Tech Stack
- Python
- JSON for storage
- File handling
- OOP design
